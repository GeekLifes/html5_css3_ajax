# 全屏幕




![前端与移动开发学院][1]
>`HTML5`支持让任意元素在页面上全屏显示

## 语法

* 基础语法

```
    requestFullScreen() 开启全屏显示
    cancleFullScreen() 关闭全屏显示 
```    
    

* 使用注意:

```
不同浏览器需要添加前缀如：
webkitRequestFullScreen、mozRequestFullScreen
webkitCancleFullScreen、mozCancleFullScreen
```
